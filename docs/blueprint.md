# **App Name**: Lab Inventory Tracker

## Core Features:

- Material Management: Add new materials to the inventory, capturing name, project, lot number, storage location, concentration, submission date, storage condition, submitted volume/mass, label information, and notes.
- Inventory Transactions: Record inventory transactions (consumption, addition, adjustment) with material reference, transaction type, quantity, unit, timestamp, and notes, automatically updating inventory quantity.
- Excel Import: Import historical inventory transactions from an Excel file, validating and normalizing data before updating balances.
- Transaction History: Shows current inventory levels along with the complete audit trail of transactions.
- Form Validation: Adds field validations and auto-completion to forms by detecting the patterns and potential field entries of common user inputs such as scientific names.

## Style Guidelines:

- Primary color: A muted blue (#6699CC) to convey reliability and precision.
- Background color: A light, desaturated off-white (#F0F4F8) for a clean and neutral backdrop.
- Accent color: A subtle green (#8FBC8F) to highlight important actions and information.
- Body and headline font: 'Inter' (sans-serif) for a clean and modern interface. It offers excellent readability, even in data-rich contexts.
- Use simple, consistent icons from a library like Font Awesome or Material Icons to represent inventory actions and material types.
- Emphasize a clear and logical layout, grouping related information and controls together for intuitive navigation and quick access to key functions.
- Employ subtle animations (e.g., transitions, feedback on interaction) to enhance usability without distracting from the task at hand.
